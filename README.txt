Petite étude de Cas sur les bases de CI avec la POO en Java
Cas 1 :
1. Définir une classe Etudiant ayant comme :
a. attributs : matricule, nom, date de naissance, liste des notes
2. Définir la classe GestionEtudiant qui fonctionne comme suit :
a. Créer une liste de matières
b. Demander l’effectif de la classe à l’utilisateur, il entre n
c. Permettre à l’utilisateur d’enregistrer ses n étudiants qui seront stockés dans un tableau
avec pour chacun ses informations et ses notes dans toutes les matières.
d. Calculer la moyenne de chaque élève
e. Afficher un menu qui permet à l’utilisateur de :
- Afficher les étudiants par ordre de mérite
- Afficher les informations sur le premier étudiant
- Afficher les informations du dernier étudiant
- Réinitialiser la liste de la classe
- Quitter du programme
